# Games-Java
2D games developed using AWT in Java. Implemented Keyboard Event Handling using Listeners.
1) Brick Out - A bouncing ball eliminates bricks. Core game physics was implemented through Collision Detection logic, which precisely manages the ball’s deflection off the paddle, walls, and the individual bricks.
2) VacuumCleaner - a moving vacuum cleaner that collects the closest stones in order. There is an option to add stones and to stop/resume the vacuum cleaner’s operation.
3) Sewerage - A matrix structure of the sewer system with the ability to add pipes rotated in different directions.
